**命令行的行为（傻瓜式直连）**： PowerShell、CMD、`wget`、`curl`、`git` 这些命令行工具，默认情况下**完全无视** Windows 的系统代理设置。

- 它们默认遵循“直连优先”的原则。
- 它们不知道你电脑上开了代理软件。
- **除非你显式地告诉它**。

所以要进行显示设置代理：

```
$env:https_proxy="http://127.0.0.1:7890"
wget.exe -c "https://huggingface.co/datasets/roneneldan/TinyStories/resolve/main/TinyStories_all_data.tar.gz"
```

下载命令

```
wget.exe -c --no-check-certificate "https://hf-mirror.com/datasets/roneneldan/TinyStories/resolve/main/TinyStories_all_data.tar.gz"
```

解压命令

```
tar -xzf TinyStories_all_data.tar.gz
```
